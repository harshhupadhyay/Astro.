# 🧥 ASTRO Winter Armor – Product Landing Page

A modern and responsive fashion product landing page built using HTML5 and CSS3. 

This project showcases a futuristic winter jacket UI inspired by premium e-commerce websites. The design focuses on clean layouts, soft shadows, modern typography, and a structured product presentation.

## ✨ Features
* **Modern E-commerce UI:** Sleek, user-friendly interface.
* **Responsive Layout:** Adapts beautifully to different screen sizes.
* **Product Showcase Section:** Highlighting the jacket with clean imagery.
* **Interactive Elements:** Size selection buttons and product color variants.
* **Shopping Controls:** Wishlist & "Add To Bag" buttons.
* **Feature Highlight Cards:** Clean callouts for product specs.
* **Visual Polish:** Soft product shadows and a clean, minimal design.

## 🛠️ Technologies Used
* **HTML5**
* **CSS3** (Flexbox & CSS Grid)
* **Google Fonts**
* **Icons:** Font Awesome & Remix Icons
